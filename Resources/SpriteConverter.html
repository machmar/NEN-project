Mask + Color PNG Interlacer

Local-only browser tool. Load two PNG files of the same size, threshold each pixel to 1-bit using its RGB average, then pack each image as a continuous row-major 1-bit stream with LSB-first bits, and interleave the resulting bytes as mask0, color0, mask1, color1, .... Pixels with average RGB value > 127 become white, otherwise black. No row padding is added; only the final byte of each stream may contain trailing zero padding.

1. Select PNG files

Waiting for input.

Validation rules: exact PNG only, same width and height for both files, per-pixel thresholding by average RGB, average RGB > 127 = white, otherwise black, black = 1, white = 0.

2. Packing rules

Pixel order: left to right, top to bottom, continuous across rows.

Bit order inside each byte: first pixel goes to bit 0 (LSB), second to bit 1, etc.

Width 9 example: pixel x=8 of the first row becomes bit 0 of the second byte.

Thresholding: each pixel uses average RGB; > 127 becomes white, otherwise black.

Output: one C array plus width/height variables and comments containing the source filenames.
